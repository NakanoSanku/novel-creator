# /novel-creator:status

此命令显示当前小说项目的详细进度。

## Claude 指令

1. **读取状态文件**：使用 `Read` 工具读取 `.novel-creator.state.json`。
2. **读取项目文件**：扫描 `./novel/` 目录下的所有文件。
3. **生成报告**：以表格形式展示：
   - 项目名称
   - 当前步骤（1-6）及步骤名称
   - 已完成的步骤列表
   - 已创建的文件统计
   - 总字数统计（如果有章节文件）
4. **进度可视化**：使用进度条或百分比显示完成度。
5. **建议下一步**：根据当前状态给出建议。

## 报告格式示例

```
📚 项目：《仙道逆天》
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
📍 当前步骤：第 4 步 - 剧情细纲
📊 整体进度：[████████░░] 66%

✅ 已完成：
  • 头脑风暴 ✓
  • 世界观设定 ✓
  • 角色与大纲 ✓

📁 项目文件：
  • 设定文件：3 个
  • 角色卡：5 个
  • 章节：0 个
  • 总字数：0 字

💡 建议：开始编写第一章的细纲
```

## 使用示例

- `/novel-creator:status` - 查看当前项目状态。
