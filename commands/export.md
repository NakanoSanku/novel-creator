# /novel-creator:export

此命令将所有章节合并导出为完整的小说文档。

## Claude 指令

1. **读取状态**：获取项目名称。
2. **扫描章节**：读取 `./novel/chapters/` 目录下的所有 `.md` 文件。
3. **按顺序排列**：根据文件名（001.md, 002.md...）排序。
4. **合并内容**：
   - 添加书名作为一级标题
   - 每个章节添加章节标题
   - 保留原有格式
5. **导出文件**：
   - 默认导出为 `./novel/export/《书名》-完整版.md`
   - 如指定 txt 格式，则导出为纯文本
6. **统计信息**：报告总章节数和总字数。

## 导出格式

### Markdown 格式
```markdown
# 《仙道逆天》

---

## 第一章 初入宗门

正文内容...

---

## 第二章 废柴之名

正文内容...
```

### 纯文本格式
```
《仙道逆天》

第一章 初入宗门

正文内容...

========================================

第二章 废柴之名

正文内容...
```

## 使用示例

- `/novel-creator:export` - 导出为 Markdown 格式。
- `/novel-creator:export txt` - 导出为纯文本格式。
