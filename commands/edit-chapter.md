# /novel-creator:edit-chapter

此命令对已有章节进行修改和润色。

## Claude 指令

1. **解析参数**：获取章节号和修改要求。
2. **读取章节**：读取对应的章节文件。
3. **分析修改需求**：
   - 如果用户指定了具体修改（如"加强对话"），按要求修改
   - 如果没有具体要求，进行通用润色检查
4. **修改类型**：
   - **润色**：改善文笔、消除冗余
   - **扩写**：增加描写、丰富细节
   - **缩写**：精简内容、加快节奏
   - **对话优化**：让对话更自然、有个性
   - **一致性检查**：与设定和角色保持一致
5. **展示对比**：显示修改前后的关键段落对比。
6. **确认保存**：询问用户是否确认修改。

## 常见修改指令

| 指令 | 说明 |
|------|------|
| 润色 | 提升文笔质量 |
| 扩写 | 增加细节描写 |
| 缩写 | 精简冗余内容 |
| 加强打脸 | 强化爽点场景 |
| 优化对话 | 让对话更有特色 |
| 增加内心戏 | 丰富心理描写 |

## 使用示例

- `/novel-creator:edit-chapter 1 润色` - 润色第一章。
- `/novel-creator:edit-chapter 3 加强打脸场景` - 强化第三章的打脸情节。
