# 场景描写指南

## 环境描写原则

### 1. 服务于情节
环境描写不是为了堆砌词藻，而是为了：
- 烘托氛围
- 暗示危险
- 展现世界观
- 引发情感共鸣

### 2. 五感法则
调动读者的五感：
- **视觉**：色彩、光影、形状
- **听觉**：声音、音乐、寂静
- **嗅觉**：气味、香气
- **触觉**：温度、质感
- **味觉**：特定场景使用

## 常见场景模板

### 宗门场景
```
青云宗坐落于九霄山之巅，云雾缭绕间，隐约可见层层叠叠的殿宇楼阁。
晨钟暮鼓之声从主峰传来，灵气如潮水般在山间流转，滋养着漫山遍野的灵药仙草。
弟子们或打坐修炼，或切磋武艺，一派仙家气象。
```

### 战斗场景
```
两道气势在虚空中碰撞，肉眼可见的气浪向四周扩散。
地面龟裂，沙石飞扬。
周围的弟子连连后退，脸色煞白——这是他们从未见过的层次。
```

### 秘境场景
```
推开那扇石门的瞬间，一股腐朽的气息扑面而来。
洞穴深处，磷火跳动，忽明忽暗。
远处似乎有什么东西在蠕动...
```

### 都市场景
```
霓虹灯在雨水中模糊成一片光晕。
深夜的酒吧街人声鼎沸，各色男女穿梭其间。
李峰靠在暗处的墙角，注视着对面大厦的顶层——目标就在那里。
```

## 动静结合

### 以静写动
```
整个战场安静得可怕。
没有人敢呼吸，没有人敢眨眼。
所有人都在等待——等待那最后一击的降临。
```

### 以动写静
```
林枫盘坐在瀑布之下，任由冰冷的水流冲刷着身体。
轰鸣的水声中，他的心却如同古井一般平静。
```

## 氛围营造

### 压抑氛围
- 使用阴暗的色调（灰、黑、暗红）
- 强调寂静、沉闷
- 使用短句，制造紧张感

### 热血氛围
- 使用明亮的色调（金、红、白）
- 强调声音、动作
- 使用排比句，制造气势

### 神秘氛围
- 使用朦胧的色调（紫、幽蓝）
- 强调未知、模糊
- 留白，让读者自行想象
