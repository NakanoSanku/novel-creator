# 力量体系设计 (Power Systems)

## 1. 经典修真体系 (Traditional Cultivation)
这是读者认知成本最低、最经典的体系。

1. **炼气期 (Qi Refining)**：引气入体，共有九层/十层。
2. **筑基期 (Foundation Establishment)**：铸就道基，寿元两百，可御剑飞行。
3. **金丹期 (Golden Core)**：体内结丹，寿元五百，称"真人"。
4. **元婴期 (Nascent Soul)**：破丹成婴，肉身毁坏元婴可逃，寿元千载，称"老祖"。
5. **化神期 (Soul Formation)**：领悟天地法则，拥有神识。
6. **炼虚期 (Void Refining)**：破碎虚空。
7. **合体期 (Integration)**：肉身与元婴融合。
8. **大乘期 (Grand Ascension)**：凡界巅峰。
9. **渡劫期 (Tribulation)**：通过雷劫飞升仙界。

## 2. 异能/等级体系 (Level/Rank Systems)

### 字母等级 (Letter Ranks)
- **F -> E -> D**：普通人/低级炮灰。
- **C -> B**：中坚力量。
- **A**：高手，一方大佬。
- **S**：国家级战略力量。
- **SS/SSS**：传说级，核弹级威慑。

- 11-20级：师
- 21-30级：大师
- ...

## 3. 力量平衡 (Balance)

### 越级挑战 (Fighting Above Rank)
主角凭什么越级打怪？必须合理化：
- **装备压制**：你有神剑，他只有铁剑。
- **功法压制**：你修神级功法（蓝条长、回血快），他修大路货。
- **属性克制**：至阳之火克制阴寒鬼物。
- **战斗智商**：利用陷阱、阵法、偷袭。
- **消耗战**：主角药多，耗死对方。

### 战力崩坏的避免 (Avoiding Scale Collapse)
- **地图限制**：高等级人物无法进入低级地图（天地法则压制）。
- **资源限制**：高级修士出手需要消耗大量珍贵资源，平时都在沉睡。
- **势力制衡**：老怪不敢随意对小辈出手，因为怕引发宗门大战（打了小的引来老的）。

## 4. 货币与资源 (Economy)
力量体系必须配套经济体系。

- **硬通货**：灵石、金币、魔晶。
- **汇率**：通常 1 上品 = 100 中品 = 10000 下品。
- **通货膨胀**：换地图时，低级货币通常贬值或被淘汰。
  - *解决方案*：在新地图，下品灵石只是地板砖，大家都在用"仙晶"。
