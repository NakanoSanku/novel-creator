# 对话写作技巧

## 基本原则

### 1. 对话要有目的
每一句对话都应该：
- 推动剧情发展
- 展现角色性格
- 传递重要信息
- 制造冲突或张力

### 2. 避免"你好我好"式对话
❌ 错误示范：
```
"你好。"
"你好，最近怎么样？"
"还好，你呢？"
"我也还好。"
```

✅ 正确示范：
```
林枫刚踏入大殿，赵天阳的冷笑声便传了过来。
"哟，这不是我们青云宗的废物弟子吗？今天怎么有脸来这里？"
```

## 角色语言特色

### 主角
- 话少但有力
- 关键时刻一语定乾坤
- 示例："你不配。"

### 傲慢反派
- 喜欢用疑问句质问
- 常用"你也配？"、"就凭你？"
- 示例："就凭你这个废物，也想和我争？"

### 忠诚配角
- 话多、热心
- 常用"老大"、"兄弟"
- 示例："兄弟，这事包在我身上！"

### 高冷女主
- 言简意赅
- 常用"嗯"、"无聊"
- 示例："...随你。"

## 对话节奏

### 快节奏（冲突场景）
```
"你找死！"
"那就来。"
话音未落，两道身影已经碰撞在一起。
```

### 慢节奏（情感场景）
```
林枫沉默了很久，才缓缓开口。
"那一天...我失去了一切。"
他的声音很轻，却让整个房间都安静了下来。
```

## 动作描写配合

对话不是孤立的，要配合动作和表情：

```
赵天阳冷笑一声，将茶杯往桌上重重一放。
"林枫，你以为你是谁？"
林枫抬眼看他，目光平静得像一潭死水。
"一个你惹不起的人。"
```

## 潜台词

高级对话要有言外之意：

```
长老意味深长地看了他一眼。
"年轻人，有些东西，不是你能碰的。"
林枫心中一凛——长老知道了。
```
