# 战斗描写技巧

## 战斗三要素

### 1. 力量对比
让读者知道双方的实力差距：
```
赵天阳是筑基后期，而林枫不过炼气七层。
在所有人看来，这根本不是一场战斗，而是一场单方面的碾压。
```

### 2. 转折点
让弱者逆袭，或让局势反转：
```
就在所有人以为林枫必死之际，他体内的龙魂突然苏醒。
一股远古的威压从他身上爆发，让赵天阳的脸色瞬间大变。
"这...这是什么力量！"
```

### 3. 结果冲击
战斗结果要有冲击力：
```
赵天阳的身体砸在地上，扬起漫天尘土。
全场死寂。
一个炼气期的废物，竟然击败了筑基后期的天才？
```

## 战斗节奏

### 快攻型
短句，连续动作，不给读者喘息：
```
出拳！
格挡！
膝击！
闪避！
林枫的攻击如暴风骤雨，不给对手任何反应的机会。
```

### 对峙型
长句，心理描写，制造紧张：
```
两人相距三丈，谁也没有动。
风吹过，卷起几片落叶。
林枫知道，接下来的一击，将决定生死。
他必须抓住那唯一的破绽。
```

## 招式描写

### 避免流水账
❌ 错误：
```
林枫使出第一招，然后使出第二招，接着使出第三招。
```

✅ 正确：
```
林枫身形暴涨，化作一道残影。
青冥剑诀的第三式——万剑归宗！
无数剑气凭空凝聚，如同雨点般向赵天阳倾泻而下。
```

### 动作具象化
- 用比喻让招式可视化
- 强调速度、力量、范围
- 配合环境反应（地面碎裂、气浪扩散）

## 群战与单挑

### 单挑：聚焦细节
```
赵天阳的拳头擦着林枫的脸颊呼啸而过，带起一道血痕。
好快！
林枫心中一凛，侧身的同时反手一掌拍向对方腰间。
```

### 群战：宏观视角
```
战场上刀光剑影，喊杀声震天。
青云宗的弟子们结成剑阵，勉强抵挡着魔修的疯狂进攻。
而在战场中央，林枫正在与三名魔尊激战。
```

## 受伤与恢复

适当的受伤描写增加真实感：
```
林枫单膝跪地，嘴角溢出一丝鲜血。
刚才那一击，震伤了他的内腑。
但他的眼神依然锐利，如同受伤的猛兽。
"就这点本事？"
```
