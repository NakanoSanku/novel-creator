# 常见问题与避坑指南 (Common Problems)

## 1. 主角光环太重 (Blatant Plot Armor)
**症状**：反派智商突然下线；主角跳崖必捡秘籍；运气好得不讲道理。
**后果**：读者觉得主角是巨婴，没有成就感。
**解决方案**：
- **交换原则/代价**：得到的机缘必须伴随着巨大的风险或代价（如：捡到魔剑，但会侵蚀神智）。
- **合理的铺垫**：主角之所以能赢，是因为之前的伏笔（比如之前救过某个小动物，这次小动物来报恩），而不是机械降神。

## 2. 战力崩坏 (Power Creep)
**症状**：前期神劲强者恐怖如斯，后期神劲不如狗。
**解决方案**：
- **不要过早透支高战力**。前期别写"毁天灭地"，就写"开碑裂石"。
- **始终保持参照系**。主角升级了，破坏力提升了，但环境（建筑材料、空间稳固度）也提升了（换了仙界地图），所以视觉效果不会无限膨胀。

## 3. 角色失踪 (Forgotten Characters)
**症状**：前期很重要的女主或兄弟，写着写着几百章没出现。
**解决方案**：
- **定期盘点**：使用 [角色卡] 检查谁很久没出场了。
- **合理的下线理由**：闭关了、被神秘势力带走了、去执行秘密任务了。
- **支线剧情**：通过传闻或书信交代他们的动向。

## 4. 甚至不需要智商的反派
**症状**：反派不仅坏，而且蠢，只会送死。
**解决方案**：
- 赋予反派**合理的利益动机**。他针对主角不是因为天生坏，而是因为主角挡了他的路，或者主角身上有他必须要的东西。
- 让反派也**进步**。主角在升级，反派也在奇遇升级。

## 5. 虐主过度 (Too Much Suffering)
**症状**：主角一直被压着打，憋屈太久，没有释放。
**解决方案**：
- **如无必要，勿增实体（虐）**。虐是为了更好的爽。
- **报仇不过夜**。小仇当场报，大仇分阶段报。不要让读者郁闷超过 3-5 章。

## 6. 说明文过多 (Info Dumping)
**症状**：一开头就用 2000 字介绍世界历史、地理、修为等级。
**解决方案**：
- **碎片化交代**。不要一次塞给读者。
- **随剧情触发**。遇到这个等级的敌人时，再借旁人之口介绍这个等级的特点。
